# Activity Shelf (Product Spec)

## Positioning

`Activity Shelf` is a gentle personal tool for collecting enjoyable things a person could do with themselves when they want comfort, play, novelty, or a small act of self-care.

It is not a task manager, planner, habit tracker, or productivity system.

The core feeling should be:

- low pressure
- visually inviting
- easy to browse
- softly supportive
- lightly self-loving

## Target User

The primary user is someone who:

- often has free time but struggles to decide what to do with it
- wants a more caring and inspiring alternative to a to-do list
- enjoys creative, cozy, leisure, reflective, or playful solo activities
- may want a small amount of personal-growth energy, but not in a rigid or self-optimizing way

They are not looking for deadlines, streaks, pressure, or performance metrics.

## Core Problem

When someone has space in their day, they often know there are many things they might enjoy, but cannot easily recall or choose one in the moment.

This leads to:

- feeling blank or indecisive
- defaulting to passive scrolling
- forgetting activities that genuinely feel nourishing or fun

## Product Promise

Activity Shelf helps users keep meaningful leisure options visible and easy to choose from, either by browsing themed shelves or by receiving a suggestion that fits their mood, energy, and available time.

## Core Actions

1. Create shelves for different kinds of activities.
2. Add personal activities to a shelf.
3. Let the app suggest activity ideas for a shelf.
4. Ask the app to recommend one activity for right now.
5. Lightly cycle activities after doing them, instead of permanently completing them.

## Main Screens

### 1. Home / Shelf Board

The main screen shows a board of shelves.

Each shelf:

- has a theme or title
- contains multiple activity ideas
- can accept new custom activities
- can receive auto-suggested activities

### 2. Suggestion Panel

A focused area on the home screen where the user can:

- choose mood
- choose energy
- choose available time
- get one suggested activity
- reroll for another option

### 3. Add Activity Flow

A lightweight modal or inline flow where the user adds:

- activity title
- mood
- energy level
- time length
- target shelf

### 4. Add Shelf Flow

A lightweight modal or inline flow where the user creates:

- shelf name
- optional visual mood or color

## MVP Features

- shelf board layout
- starter shelves with example activities
- custom shelf creation
- custom activity creation
- browser local storage persistence
- auto-suggest ideas for selected shelves
- suggestion engine based on mood, energy, and time
- reroll suggestion
- hide current suggestion item
- mark activity as done and move it to the back of the shelf

## Product Rules

- no deadlines
- no due dates
- no chores
- no work-task framing
- no guilt language
- no heavy productivity metrics
- no streaks in MVP

## Design Direction

- warm, airy, slightly dreamy
- tactile card-like shelves
- inviting rather than efficient-looking
- expressive typography
- gentle language instead of command language

## Future Features

- favorites or comfort picks
- richer auto-suggestion generation
- tags such as indoor, outdoor, quiet, messy, screen-free
- image covers or stickers per shelf
- seasonal shelves
- journal note after doing an activity
- shareable shelf templates
